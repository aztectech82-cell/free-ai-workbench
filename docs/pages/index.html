Free & Open Source

Run AI on your computer.
No subscription. No cloud. No cost.

Free AI Workbench gives you a clean, click-first interface for local AI models -- DeepSeek, Qwen, Mistral -- running privately on your own machine.

Get Started on GitHub View Installer

One-command Windows install -- open PowerShell and run:

irm https://raw.githubusercontent.com/aztectech82-cell/free-ai-workbench/main/installer/install-free-ai-workbench.ps1 | iex
💬

Chat with Local Models

Pick any installed Ollama model. Ask One for a single response or Ask All to compare every model simultaneously.

🖼️

Screenshot Context

Paste a screenshot with Ctrl+V. Ask a vision model what is on screen. Works with Qwen Vision and Moondream.

📁

File Actions

Read, create, edit, or append local files using AI. Your files stay on your machine -- nothing leaves your computer.

🎯

AI Framing Presets

Business, Finance, Engineering, Marketing, Writing, Research, Career, and more. One click loads a purpose-built AI context.

🔑

Optional API Keys

Add OpenAI, Anthropic, DeepSeek, or Groq keys anytime. They unlock cloud models when local isn't enough -- but you never need them to start.

🖥️

Desktop Shortcut

The installer creates a desktop shortcut. Double-click to launch. No terminal, no commands, no setup every time.
